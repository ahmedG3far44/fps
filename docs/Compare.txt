# Feature: Game Price Comparison (Official Stores & Authorized Resellers)

Implement a complete "Game Price Comparison" feature for the FrameWise AI platform.

The purpose of this feature is to allow users to see the cheapest legitimate place to buy a game directly from the Game Details page.

We ONLY support:

## Official Stores

- Steam
- Epic Games Store
- EA App
- Ubisoft Connect
- GOG
- Battle.net
- Microsoft Store
- Rockstar Launcher

## Authorized Resellers

- Humble Bundle
- Fanatical
- Green Man Gaming
- Gamesplanet

DO NOT include:

- G2A
- CDKeys
- Kinguin
- Eneba
- Any gray-market key seller

This feature should only appear for PAID games.

If a game is free, the section should not render.

# Data Provider

Use IsThereAnyDeal API.

Documentation:
https://docs.isthereanydeal.com/

Use ONLY legitimate stores and official reseller data.

The API already excludes gray-market stores and is intended for official deals and authorized sellers.

Never scrape websites.

Never use unofficial APIs.

Use IsThereAnyDeal as the single source of truth.

---

# Feature Requirements

When the user opens:

/games/[slug]

Fetch all available prices for that game.

Show:

- Store name
- Store logo
- Current price
- Original price
- Discount percentage
- Currency
- Buy URL
- Official or Authorized badge
- Best Deal indicator

Sort ascending by price.

The cheapest available offer must appear first.

# Database Schema

Create:

Store
GameStorePrice

Prisma:

model Store {
  id           String @id @default(cuid())
  name         String
  slug         String @unique
  logoUrl      String?
  type         StoreType

  prices       GameStorePrice[]
}

enum StoreType {
  OFFICIAL
  AUTHORIZED_RESELLER
}

model GameStorePrice {
  id            String   @id @default(cuid())

  gameId        String
  storeId       String

  countryCode   String
  currency      String

  currentPrice  Decimal
  originalPrice Decimal?

  discount      Int?

  buyUrl        String

  updatedAt     DateTime @updatedAt

  game          Game @relation(fields:[gameId], references:[id])
  store         Store @relation(fields:[storeId], references:[id])

  @@index([gameId])
  @@index([countryCode])
}
